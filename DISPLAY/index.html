Long read scroll

DESCRIPCIÓN

Una web con un contenido largo que se despliegue a medida que haces scroll. En este caso una web que cuenta dos historias de halloween.

Puntos destacados

  • Estructura: dos secciones principales, cada una con una historia. Cada historia dividida en capitulos.
  • Al ser un cuento de halloween los colores son apagados y oscuros.
  • Se incorporan elementos animados como divs con textos en hover y las imagenes con java script.

Microsite

DESCRIPCIÓN

Una web en la que no se pueda hacer scroll, divida en varias subpáginas que resuelvan un problema local. En este caso "EcoMadrid" una web que te ayuda a reciclar y cuidar el planeta.

Puntos destacados

  • Estructura: Consta de una página pricipal y tres subpáginas. Una con ubicaciones de puntos limpios, otra con posibles soluciones a problemas de reciclaje y otra con varias formas de contacto a la organización.
  • Incluye colores neutros predominando el uso del verde al tratarse de una web medioambiental.
  • En el header se encuentran cuatro links a todas las páginas de la web.
  • Incluye animaciones en hover de cambio de colores, opacidad y movimiento de elementos.

Fronted

DESCRIPCIÓN

Una web fronted de un comercio electrónico que se divida en un menu, productos especiales y productos. Incorporando bootstrap para organizar el tamaño de la información y utilizando la clase bootstrap card. En este caso "Lumina" es una web de articulos de belleza.

Puntos destacados

  • Estructura: Un menú con posibles páginas de más productos. Dos productos destacados del apartado "Cuidado facial" y cards con más productos de la sección, sus respectivas descripciones, precio y botón para añadir al carrito.
  • Los colores siguen la estética de la marc ay las fotografías. Rosa y granate.
  • Incorpora animaciones de hover de cambio de color y bootstrap para organizar las divs de cada apartado de la web.

Landing

DESCRIPCIÓN

Una web landing page de un servicio/producto. En este caso "Archea" es una web que lanza un producto limitasdo llamado "ovola", cuyos primeros 100 compradores obtendrán un descuento en su siguiente compra.

Puntos destacados

  • Estructura: En una misma página dividida en sections según la información que ofrecen. Presentación del producto, modelos, materiales y un formulario para suscribirse.
  • Los colores siguen la estetica de la marca, limpios y neutros excepto los divs que buscan llamar la atención del cliente.
  • Incluye animaciones en hover como cambios de colores, un carrousel de fotos, videos en gif y un formulario enlazado a formspree. Además el header tiene cuatro links que te llevan a las seciones de la página.
